Introduction

Macro is a TAP that executes one or more keys for each macro key defined. Invocation of a macro can be through a single remote control (RC) key, or a combination of a “function” key (FKey) together with a RC key (e.g. similar to Alt TAB on a PC – here Alt is the FKey and TAB is the RC key). The use of the FKey can be varied to be

·        always on, i.e. No FKey is required, or

·        use key once, i.e. press the FKey once to enable the macros and from then on simply press the RC key to invoke the macro, or

·        use key every time, i.e. you must press the FKey with the RC key to invoke the macro

Installation

The Macro TAP is managed using the TSR Commander TAP or by defining the Config key in the [macro] section. You can download the TSR Commander TAP from here: http://tonyspage.abock.de/

 

The Macro TAP can be installed in TSR mode – if so place it in your Auto Start folder – or start it manually. This can be done using the Topfield Altair tool and a USB cable. The TAP stores the macros and any customisations in its own ini file (macro.ini) which must be in the ProgramFiles directory. However, the TAP will work without any ini file and you can create all customisations via the gui (config macro).

Usage

The TAP’s behaviour is modified through options in the file macro.ini or through the configuration screen. Modifying options through the configuration screen only last while the Macro TAP is running unless you save them to file, which updates the macro.ini file.

The file macro.ini should contain up to 3 sections:

·         [macro] – contains key redefinitions for the TAP as a whole (not the macros). For example, the key to stop the TAP, select OK , the function key, etc. In the following sections “RC key” is any key defined in the file KEY.H (included in the distribution). Syntax is:

o        <keyword>=<RC key>

<keyword> can be:

·         ExitTAP         - key to exit the TAP

·         Function         - the FKey

·         Config            - key to bring up the config screen

·         Up                 - key to move upwards in the menu

·         Down             - key to move down in the menu

·         Confirm          - key to confirm selection in the menu

·         Delete            - key to delete items in the menu

For example:

 

ExitTAP=RKEY_Exit

 

If you use any FunctionState (see below) other than AlwaysOn, you must have one line that defines your function key as follows:

 

Function=<RC Key>   where <RC Key> is any remote control key.

 

·         [options] – contains the options that dictate how the macros will work.

o        FunctionState – this can be one of:

§         AlwaysOn (default) - no need to press a function key to activate macros

§         UseKeyOnce - press the function key once and macros are enabled from then on

§         UseKeyEveryTime - function key must be pressed every time a macro is required

o        FunctionKeyTime – maximum number of clicks to wait between the FKey and the <key to be redefined>. A click is 10ms (ie. 100 clicks = 1 second)

o        LoggingLevel – one of LogNone|LogSome|LogMore|LogAll - LogAll goes to Serial port also. Logging writes output to the file ProgramFile\macro.log

 

·         [function keys] – contains one line for each macro assignment. Syntax is:

o        <key to be redefined>=<macro list>

o        <key to be redefined> is any one RC key defined in KEY.H (included), e.g. RKEY_Sleep.

o        <macro list> is 0 or more items separated by commas. Items can be RC keys found in KEY.H, a delay option, or a menu option.

 

For example to redefine the Sleep key to record add the following line:

RKEY_Sleep=RKEY_Record

To redefine the Sleep key to go up 2 channels add the following line:

RKEY_Sleep=RKEY_ChUp,RKEY_ChUp

To disable the Sleep key add the following line:

RKEY_Sleep=

To go to the Timers menu you can do the following using the Sleep key:

RKEY_Sleep=RKEY_Menu,D100,M1,D100,RKEY_Ok,D100,M2,D100,RKEY_Ok

 

Example

;this is a sample ini file for the macro tap
[macro]
ExitTAP=RKEY_Exit
Function=RKEY_Recall

[function keys]
RKEY_9=RKEY_1,D100,RKEY_0

; options section:
[options]
FunctionState=UseKeyEveryTime
ExitMenu=N
FunctionKeyTime=200
LoggingLevel=LogAll
