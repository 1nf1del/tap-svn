MakeLogos 1.0
----------------

MakeLogos is a command line replacement for the MakeLogos TAP.

MakeGraphics can process gif, png, jpeg and pcx files and will output a logo.dat file for use with UKTimers, Archive, Surfer or AutoSearch.

Examples
--------
Process all png files
makegraphics *.png

Process all gif and pcx files with a name beginning UI and output backup_logo.dat
makegraphics -g -obackup_logo.dat *.gif UI*.pcx
