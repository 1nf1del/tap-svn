TAP Project - Documentation

The project documentation is broken down into smaller modules designed to cover particular topics. Some modules provide step by step instructions, others cover things in more generic terms.

Mailing lists

Optional, but highly recommended: If you have not already done so, subscribe to the project mailing lists:

TAP-announce for project related announcements. This is a moderated, low traffic list.
TAP-devel for developer talk. Subscribers can post to this list to collaborate.

Text editor

You will need a decent text editor suitable for programming. Your editor needs to support UNIX line endings. Windows Notepad will not do.

There are several very powerful text editors, such as vim and emacs, available on most UNIX systems and in Cygwin. Windows and Mac users may find these editors a bit hard to use initially, however there are plenty of alternative editors available. Windows users may wish to take a look at Programmers Notepad 2, Mac users can check out Smultron.

Compiler (toolchain) installation

In order to compiler TAPs, you need to install the compiler, linker, header files and other utilitites. This is also known as the toolchain.

The toolchain can be downloaded from the download section of the project. There are binaries available for several platforms. These are intended to be installed in /usr/local/topfield-gcc
