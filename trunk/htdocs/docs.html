TAP Project - Documentation

The project documentation is broken down into smaller modules designed to cover particular topics. Some modules provide step by step instructions, others cover things in more generic terms.

  1. Optional, but highly recommended: If you have not already done so, subscribe to the project mailing lists:
    TAP-announce for project related announcements. This is a moderated, low traffic list.
    TAP-devel for developer talk. Subscribers can post to this list to collaborate.

  2. Cygwin setup.

    Only relevant to those using Windows computers. Explains how to install and configure Cygwin - the UNIX compatibility environment.

  3. Initial TAP project environment setup.

    This is relevant to everyone. Explains how to create the required directories and retrieve all the current source code. Also explains how to create your own project and use the Makefile based build infrastructure to have your TAPs compiled automatically.
