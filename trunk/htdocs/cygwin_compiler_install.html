TAP Project - Cygwin TAP compiler install

Here are step-by-step instructions for installing the TAP compiler in a Windows Cygwin environment.
  1. Double-click your TAP shell icon to start the Cygwin shell and get a shell prompt.
  2. Download the compiler binaries. 
    username@computer ~
