TAP Project

This project endeavours to foster collaboration between the developers of TAPs. TAPs are third party programs that can be installed on a Topfield PVR.

Main areas of development:
  • TAPs
  • reusable code libraries for inclusion in TAPs
  • tools for development and management of TAPs
